Como Jogar

Guie o pacote de dados da memória mais lenta até a CPU. Cada zona tem velocidade e obstáculos diferentes!

HD/SSD — Zona Lenta

Paredes móveis (Fragmentação). Velocidade: 0.5x

RAM — Zona Média

Buracos de Page Fault. Velocidade: 1.2x

Cache L3/L2 — Rápido

Projéteis de Cache Miss. Velocidade: 2x

Cache L1 — Muito Rápido

Obstáculos densos. Velocidade: 3x

CPU — Objetivo!
